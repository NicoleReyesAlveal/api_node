
First install NodeJS:
`$ brew install node`


After cloning the repository,  install these dependencies
```
  "dependencies": {
    "body-parser": "^1.19.0",
    "express": "^4.17.1",
    "mongodb": "^3.6.6",
    "mongoose": "^5.12.8"
  }
  ```
running the following command: 
`$ npm install <dependency_name>`


To run the server:
`$ node app.js`
