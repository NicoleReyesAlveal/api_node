
First install NodeJS: `$ brew install node`

After cloning the repository,  install these dependencies
```javascript
  "dependencies": {
    "body-parser": "^1.19.0",
    "ejs": "^3.1.6",
    "express": "^4.17.1",
    "mongodb": "^3.6.6",
    "mongoose": "^5.12.8"
  }
  ```
Run the following command: 
`$ node app.js`
